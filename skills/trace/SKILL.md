# trace

Record source, target, files/assets, verification, persistence, and coordinate summary. Use the finish command or `python .coderail/coderail.py trace-index`.

## Rules

- Keep output concise.
- Prefer G/T/S/V/X/P over duplicated fields.
- Do not write business code unless this skill explicitly covers execution.
- Do not hide failed verification.
