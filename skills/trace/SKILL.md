# trace

Record source, target, files/assets, verification, persistence, and coordinate summary. Use scripts/trace_event.py, then scripts/trace_index.py.

## Rules

- Keep output concise.
- Prefer G/T/S/V/X/P over duplicated fields.
- Do not write business code unless this skill explicitly covers execution.
- Do not hide failed verification.
