# Blueprint Standard

Optional awareness reference for diagrams. Keep optional.

### System Architecture
Use when the project has multiple components.

### Component Diagram
Use when there are multiple modules.

### Sequence Diagram
Use for cross-component interactions.

### State Machine
Use for status transitions.

### ER Diagram
Use for relational data.

### User Journey Map
Use for user-facing flows.

### User Flow
Use for interaction branches.

### Page Flow
Use for page navigation.

### Data Flow Diagram
Use for data movement.

### Deployment Diagram
Use for deployment topology.

### CI/CD Pipeline
Use for release automation.
